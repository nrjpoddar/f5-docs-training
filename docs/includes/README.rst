Includes
========

This directory -- ``includes/`` contains files which demonstrate how F5
content should be structured. The following content types are used:

    * topic - feature document, uses the :ref:`feature template`
    * concept - background information or explanation; generally used to preface topics in a larger guide
    * ref - data, tables, images, etc. (i.e., reference material)
